
# US States Game


#### This is a trivia game that uses the turtle module.<br>Can you guess all 50 states of the US?



## Requirements
- Pandas
- Turtle


### To install pandas:

```
pip install pandas
```
### How to run the file

open the terminal and navigate to the path of the file
```
python main.py

```
