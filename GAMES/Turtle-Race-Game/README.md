# Turtle-Race

### A randomly simulated turtle race! You can watch different color turtles race across the screen and keep track of the wins/losses of each turtle.


## Requirements
- turtle

### How to install
- open the cmd
- navigate to the file directory
- do a pip install turtle
- python main.py
